# Changelog

### added
- added Reinforced durability system (`Reinforced I/III/V`)
- added aw_iron and raw_gold for iron/gold ore drops
- added obsidian tools with reinforced being the priority of these tools, theyre as slow as stone tho but hella tough

### Changed
- armor effect tradeoffs:
  - copper burn-time reduction consumes durability
  - gold fall mitigation now consumes durability
  - leather speed consumes durability over time only while moving
- combat:
  - copper sword ignite fixed, a 35% proc chance

- tools:
  - iron and diamond tool durability/traits retuned with reinforced 
  - you now need a copper pickaxe to mine iron ore
- food:
  - hunger/saturation-style metadata removed
  - poison potato and pufferfish now apply self-damage

### removed
- removed chicken egg-laying
- removed raw copper block

### Fixed
- fixed copper sword fire proc failing
- reduced snowball damage
